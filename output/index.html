Scar: Sun cloud API reference.

This website is generated (nanoc3) from content in the Scar repository. The aim of the Scar project is to facilitate participation by increasing

  • Agility

  • Transparency

  • Productivity

  • Generate documentation from the API specification (Ruby). Projects implmenting Sun's Cloud API can share a common documentation source and generate (possibly filtered) documentation that is extracted from the specification, and evolves as the specification is corrected or extended.

  • Provide snapshots of Sun's Cloud API. Example To grab version 0.n

    git clone git://github.com/hedgehog/scar.git git checkout -b your_branch v0.n
    If you don't want the whole repo, github is not configured to respond to git archive ... Instead, use curl -O -L http://github.com/hedgehog/scar/downloads/...

  • Make transparent, and accessible, any ideas others explore or errors they correct. See here.

  • Provide a web queryable (JSON? YAML?) representation of the API. (TODO)

  • Provide a local copy of the Sun cloud API specification that can be browsed (exactly these pages), and customized. See (TODO).

  • Change this page’s content by editing “content.html” file in the “content” directory. This is the actual page content, and therefore doesn’t include the header, sidebar or style information (those are part of the layout).

  • Change the layout , which is the “default.txt” file in the “layouts/default” directory, and create something unique (and hopefully less bland).

If you need any help with customizing your nanoc web site, be sure to check out the documentation (see sidebar), and be sure to subscribe to the discussion group (also see sidebar). Enjoy!
